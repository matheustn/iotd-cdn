# IOTD CDN — Idle of the Dead (Steam AppID 4927420)

Static asset host for **Steam Inventory Service item icons** (`icon_url` / `icon_url_large`).
Steam downloads and caches these images — URLs must stay stable forever.
**Do not delete or rename published files**; to change art, add a NEW filename and update the itemdefs.

Served via GitHub Pages: `https://matheustn.github.io/iotd-cdn/iotd/items/<file>.png`
